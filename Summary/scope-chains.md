* 首先来讲讲一些概念的东东
 * 书上是如此说的： 作用域，作用域链，闭包，垃圾回收，他们都是非常严谨的
 * var 表示一个变量是当前词法的作用域
 
 ```javascript
   function someFunc() {
      var aVariable; //只能在这个函数里使用
    }
  ```
  
  * 作用域链
  
  ```javascript
  function someFunc() {
      function inner() {
      } //inner函数镶嵌在somefunc这个函数中
    }
  ```
  
    * 所有潜逃作用域都遵循相同的规则，就是内部函数可以访问外部函数，但是外部不可以访问内部函数
