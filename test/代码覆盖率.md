#### istanbul

* 这是一个npm 模块，用于测试代码覆盖率
* 首先写一个js文件，来测试一下
  ```javascript
  function my(a) {
   console.log(a);
   }
   my(1);
   ```
* 运行命令
```
$ istanbul cover app.js   //将会返回代码覆盖率的结果，100%
```
* 为覆盖率设置门槛
 
当然，代码不一定是百分之百覆盖，所以就会出现其他百分比情况，这个时候，我们只要设置这个百分比接近某个比例就可以了
```
$ istanbul check-coverage --statement 90   //将会显示覆盖率没有超过90%的代码
```
